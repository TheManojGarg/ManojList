ManojList

List of allow, block and ignore requests to third party websites which are irrelevant to content of the website for the end user.

Download this project as a .zip file Download this project as a tar.gz file

Objective

To provide a safe and lean web experience

Desired Outcome

To achieve the stated objective, we'll focus on four major areas of the websites. First is anti-tracking, second is performance, third is transparency and the fourth is blocking irreverent content and no moral policing.

Why not Disconnect, Adblock Plus, Ghostery and similar products

Reverse approach, no "allowed" ads/tracking and transparent.
