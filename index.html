ManojList

List of allowed and blocked http requests.

// Download this project as a .zip file

Objective

To provide a safe and lean web experience

Desired Outcome

To achieve the stated objective, we'll focus on four major areas of the websites. First is anti-tracking (for privacy through blocking requests to tracking servers), second is performance (blocking irrelevant http requests and not downloading and running content like scripts etc. from third party servers), third is transparency (see all http requests)

How this is different from Disconnect, Adblock Plus, Ghostery etc.

  • Reverse approach: Instead of selecting which content to block, here we'll block every third party http requests and then just allowing selective requests to be made which are needed for user experience
  • No "allowed" ads/tracking
  • Will be transparent to the user in real time
  • Will have better performance as third party servers won't even be sent requests
  • This is not ad blocking software like it won't block google content on google websites, facebook content on facebook etc. though it'll filter a lot of third party ads like Google ads on non-Google websites
