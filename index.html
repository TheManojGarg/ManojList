ManojList

List of allow, block and ignore requests to third party websites which are irrelevant to content of the website for the end user.

Download this project as a .zip file Download this project as a tar.gz file

Objective

To provide a safe and lean web experience

Desired Outcome

To achieve the stated objective, we'll focus on four major areas of the websites. First is anti-tracking (for privacy), second is performance (blocking irreverent http requests and not runnning irrelevant scripts from third parties), third is transparency (see all http requests)

How this is different from Disconnect, Adblock Plus, Ghostery etc.

  • Reverse approach: Instead of selecting which content to block, here we'll block everything and just let selective content to be downloaded from the relevant and intended servers
  • No "allowed" ads/tracking
  • Will be transparent to the user in real time
  • Will have better performance as third party servers won't even be sent requests
  • This is not ad blocking software like it won't block google content on google websites, facebook content on facebook etc. though it'll filter a lot of theird party ads like Google ads on non-Google websites
